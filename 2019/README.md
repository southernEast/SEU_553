## 5. 包含多态

继承关系 a0 -> B -> D

由于 A0.print() 为纯虚函数，p = &b; p.print() 调用的是 B 的 print()

由于 A0、B、D 的 print() 参数、返回值类型都相同，则满足虚函数继承条件，B、D 自动被声明为虚函数

同理，B.print() 为虚函数，p = &d; p.print() 调用的是 D 的 print()

```cpp
D::print
B::print
```
