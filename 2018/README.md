## [Problem 3](3/3.cpp)
> 如果矩阵A中存在这样的一个元素A[i, j]，满足条件：A[i, j]是第i行中最小的元素，且又是第j列中最大的元素，则称之为该矩阵的一个马鞍点，请编程计算出m*n矩阵A的所有马鞍点。

> 样例输入：

```
3 3
1  7  3
5  4  6
17 18 9
```
> 样例输出：

```
(3,3) 9
```
